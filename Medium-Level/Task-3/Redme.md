# To-Do List Application

## Description
This project is a simple To-Do List Application created as part of my Web Development Internship. It allows users to add and remove tasks dynamically using JavaScript.

## Features
- Add new tasks
- Remove completed tasks
- Interactive user interface
- Dynamic task management

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Add tasks using the input box and remove them when completed.

## Project Outcome
An interactive To-Do List application that allows users to add and remove tasks efficiently.
