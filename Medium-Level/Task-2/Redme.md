# Form Validation Using JavaScript

## Description
This project is a simple Contact Form with JavaScript Validation created as part of my Web Development Internship. It validates user input before the form is submitted to ensure all required fields are filled correctly.

## Features
- Name field validation
- Email format validation
- Message field validation
- Displays error messages for invalid inputs
- Prevents form submission until all fields are valid

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Fill in the form and test the validation.

## Project Outcome
A contact form that validates user input and prevents submission if the required information is missing or invalid.
