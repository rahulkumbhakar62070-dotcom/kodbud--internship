# Contact Form

## Description
This project is a simple Contact Form created as part of my Web Development Internship. It allows users to enter their name, email, and message through a clean and organized form.

## Features
- Name input field
- Email input field
- Message textarea
- Submit button
- Clean and responsive design

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

## Project Outcome
A webpage that allows users to enter their information through a contact form.
