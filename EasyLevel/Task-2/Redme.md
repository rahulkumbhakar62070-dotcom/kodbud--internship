# Navigation Menu

## Description
This project is a simple multi-page website created as part of my Web Development Internship. It contains Home, About, and Contact pages connected through a navigation menu.

## Features
- Home, About and Contact pages
- Navigation bar on every page
- Easy page navigation
- Clean and simple design

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Use the navigation menu to switch between pages.

## Project Outcome
A simple multi-page website where users can navigate between different pages using a navigation menu.
