# Image Gallery

## Description
This project is a simple Image Gallery created as part of my Web Development Internship. It displays multiple images in a clean grid layout using HTML and CSS.

## Features
- Grid-based image gallery
- Organized image layout
- Responsive design
- Simple and clean user interface

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

## Project Outcome
A webpage that displays multiple images neatly arranged in a grid-style gallery.
